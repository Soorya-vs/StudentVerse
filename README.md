# StudentVerse – College Students Community & Placement Management Platform

## Overview

StudentVerse is a full-stack web application developed to streamline communication, collaboration, and placement-related activities within a college environment.

The platform provides a centralized space where students can join communities, receive important notices, participate in events, explore placement opportunities, and manage their profiles. Administrators can efficiently manage students, communities, notices, events, and placements through a dedicated admin dashboard.


## Features

### Student Module

* Student Registration and Login
* Profile Management
* Community Participation
* View Notices and Announcements
* View Upcoming Events
* Explore Placement Opportunities
* Community Updates Feed

### Admin Module

* Secure Admin Login

* Community Management
* Notice Management

## Key Functionalities

### Community Management

Students can join various college communities and access community-specific updates.

### Notice Management

Administrators can publish notices with images and external links.

### Event Management

Students can view upcoming events and access registration links.

### Placement Portal

Placement opportunities can be posted with company details, eligibility criteria, deadlines, and application links.

### Student Approval Workflow

New registrations require administrator approval before platform access is granted.
